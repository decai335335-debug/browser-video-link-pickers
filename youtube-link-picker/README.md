# YouTube Link Picker

一个用于 YouTube 搜索结果、播放列表、频道视频页的 Chrome MV3 插件。它会给页面上的视频卡片加选择圆点，并提供批量复制链接的浮动工具栏。

## 功能

- 自动识别 YouTube 视频链接，复制格式为 `https://www.youtube.com/watch?v=视频ID`。
- 点击视频封面：只选择或取消当前视频。
- 点击视频左上角圆点：只选择或取消当前视频。
- 点击视频标题：正常打开视频页面。
- 支持复制已选视频链接，每行一个链接。
- 页面滚动或 YouTube 异步加载新内容后会自动重新识别。

## 安装

1. 打开 Chrome 扩展程序页面：`chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择这个目录：`E:\git项目\Codex\youtube-link-picker`。
5. 打开 YouTube 搜索结果页，例如 `https://www.youtube.com/results?search_query=ai+workflow`。

## 使用

页面右下角会出现工具栏。

- 点击视频封面或左上角圆点：只选择或取消这一条。
- 点击“复制链接”：复制所有已选链接。
- 点击“全选本页”：选择当前已加载出来的所有视频。
- 点击“清空”：取消所有选择。

如果结果很多，需要先向下滚动让 YouTube 加载更多条目；插件只能复制当前页面 DOM 中已经加载的视频。
