# Bilibili Favorite Link Picker

一个用于 B 站收藏夹和视频列表页面的 Chrome MV3 插件。它会在识别到的视频卡片左上角加选择圆点，并提供批量复制链接的浮动工具栏。

## 功能

- 自动识别当前页面中的 B 站视频链接。
- 点击视频封面或预览区域：只选择或取消当前视频。
- 点击视频左上角圆点：只选择或取消当前视频。
- 点击视频标题：正常打开视频页面。
- 支持复制已选视频链接，每行一个链接。
- 页面滚动或 B 站异步加载新内容后会自动重新识别。

## 安装

1. 打开 Chrome 的扩展程序页面：`chrome://extensions/`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择这个目录：`E:\git项目\Codex\bilibili-fav-link-picker`。
5. 打开 B 站收藏夹页面，例如 `https://space.bilibili.com/你的UID/favlist`。

## 使用

页面右下角会出现工具栏。

- 点击视频封面或左上角圆点：只选择或取消这一条。
- 点击“复制链接”：复制所有已选链接。
- 点击“全选本页”：选择当前已加载出来的所有视频。
- 点击“清空”：取消所有选择。

如果收藏夹很多，需要先滚动页面让 B 站把更多视频加载出来；插件只能复制当前页面 DOM 中已经加载的视频。

## 说明

B 站页面结构可能变化，所以识别逻辑使用了比较宽松的选择器：只要页面上有指向 `bilibili.com/video/BV...` 的链接，插件就会尝试把它当作视频条目处理。
